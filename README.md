# Event Management and Ticketing System

A full-stack Event Management and Ticketing System built using **Spring Boot, React, and MongoDB**. The application enables organizers to create and manage events while allowing users to browse events, book tickets, and track seat availability in real time.

## Features

- User registration and login with JWT authentication
- Event creation, update, and deletion
- Real-time seat availability tracking
- Secure ticket booking system
- RESTful APIs for event and booking management
- MongoDB integration for data persistence
- Responsive and user-friendly interface
